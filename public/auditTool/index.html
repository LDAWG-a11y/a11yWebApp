Generated text

The 4 sections that are generated, each have a Copy button, the label of that button makes it clear what will be copied to a user's clipboard. There is an ARIA announcement set to assertive for this, presently that messages clears after 1 second, so clicking any button again will require a 1 second gap to hear an announcement.

The 2 tables for Level A and AA are presentational tables, they're designed to be copied and then pasted directly into a standard VPAT table in a Word document. The column headers and row headers have been intentionally omitted, as the intent is to highlight all the data cells in a Word template's VPAT table and then populate with this data
